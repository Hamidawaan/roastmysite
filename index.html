Overview

RoastMySite ("the Extension", "we", "us") is a free Chrome browser extension that allows users to anonymously rate and review any website they visit. This Privacy Policy explains what information we access, how it is used, and your rights as a user.

By installing and using the Extension, you agree to the practices described in this policy. If you do not agree, you may uninstall the Extension at any time from your Chrome browser.

📦
What Data We Collect

We collect NO personally identifiable information.

The Extension accesses only the following data, and only when you open the extension popup:

  • Current tab URL — read temporarily to identify which website is being reviewed. This URL is used only to fetch and display community feedback for that domain. It is never stored against your identity.
  • Anonymous feedback — the star rating (1–5), selected tags (e.g. "Design Disaster", "Slow AF"), and optional comment text that you voluntarily submit. This data contains no personal information and is stored with no link to any user account, device, or identity.

🚫
What We Do NOT Collect

We never collect, store, or transmit the following:

  • Names, email addresses, or any personal identification
  • Passwords or authentication credentials
  • Browsing history or list of visited websites
  • Page content, text, images, or DOM data from websites you visit
  • Location data — GPS coordinates or IP-based location
  • Financial or payment information
  • Health or medical information
  • Clicks, keystrokes, mouse movements, or behavioural tracking
  • Device identifiers or hardware information

⚙️
Permissions Explained

The Extension requests the following Chrome permissions. Here is exactly why each one is needed:

Permission Why it is needed
activeTab Reads the URL of the currently active browser tab only. This is used to look up community feedback for that specific website domain. No page content, cookies, or any other tab data is accessed.
storage Caches recently fetched community feedback data on your local device. This improves loading speed when you reopen the extension on a previously visited site. No personal information is stored — only anonymous ratings and tags for website domains.

🔄
How We Use Data

The anonymous feedback you submit (star rating, tags, optional comment) is stored on our servers and displayed publicly as aggregated community scores in the Extension's Feedback tab. No user identity is ever associated with submitted feedback.

We use this anonymised, aggregated data solely to display community ratings to other users of the Extension. We do not analyse, sell, or process it for any other purpose.

🤝
Data Sharing

We do not sell, rent, share, or transfer any data — personal or anonymous — to third parties for any purpose whatsoever. Anonymous aggregated feedback (e.g. "this site has a 4.2 average rating") is displayed publicly within the Extension, but it contains no user-identifying information.

💾
Local Storage

The Extension uses Chrome's built-in chrome.storage.local API to cache recently fetched community feedback on your device. This data consists only of anonymous website ratings and tags — never personal information.

You can clear this cached data at any time by removing the Extension from Chrome, or by clearing your Chrome extension data in browser settings.

👶
Children's Privacy

This Extension is not directed at children under the age of 13. We do not knowingly collect any information from children. If you believe a child has submitted feedback through this Extension, please contact us and we will remove it immediately.
